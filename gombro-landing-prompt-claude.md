# Gombro — Landing Page Prompt for Stitch

## Stack
- React + Tailwind CSS
- Mobile-first, responsive
- No animations library — CSS transitions only

---

## Concept

A literary, typographic-forward landing page. Feels like an editorial magazine crossed with a developer tool. No hero illustrations, no gradients, no fluff. Typography carries the page.

---

## Hero

```
Gombro
Inventive writing.
```

- App name in large serif display font
- Tagline directly below, lighter weight, same family
- Single CTA: **"Download for Mac"**
- No subheadline, no extra copy — let the silence work

---

## Features Section

Three cards, minimal layout. Label + one-line description only.

| Feature | Description |
|---|---|
| Bilingual toggle | Switch between EN and ES on the fly — no reinstalls, no global settings |
| Safe autosave | Atomic paragraph saving. Your words are never overwritten by a bad undo |
| Paragraph search | Find any term and jump straight to editable paragraphs with live highlight |
| Sentence mode | Rhythmic writing flow — `.` closes a paragraph, `Enter` adds spacing |
| DOCX export | One-click export per project to a clean folder structure |

## Tone & Style

- **Fonts**: serif display (Libre Baskerville or similar) + monospace accents (DM Mono or similar) for UI labels and commands
- **Color**: near-black text on off-white/warm paper background. One accent color, used sparingly
- **Layout**: generous whitespace, left-aligned, asymmetric sections
- **No**: gradients, hero images, stock photos, rounded blobs, purple-on-white
- **Voice**: confident, spare, literary — never salesy

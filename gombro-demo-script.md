# Gombro — Demo Script

## 1 · Title (0–4s)
Black screen. "Gombro" appears letter by letter, typewriter style.

## 2 · The 3 panels (4–10s)
App opens. Three panels visible:
Explorer on the left, Editor in the center, Terminal at the bottom.

## 3 · Zen mode (10–16s)
Escape. Side panels disappear. Only the editor remains, clean.

## 4 · Toggle (16–22s)
Escape again. Panels come back.

## 5 · Editor (22–28s)
User types a paragraph.

## 6 · Sentence mode (28–34s)
Sentence mode activated. Sentences separate visually.

## 7 · Baraja (34–40s)
Right-click on a paragraph → Baraja.
The paragraph transforms with material from another session.

## 8 · Reveal naipe (40–46s)
Panel "✂ revelar naipe" appears.
Click → flash in Explorer showing the source session.

## 9 · Abarajar (46–52s)
Right-click → Abarajar. The paragraph reorganizes itself.

## 10 · Search (52–58s)
: → search writing → results highlighted across sessions.

## 11 · Phrase search (58–64s)
search writing in phrases → results at sentence level.

## 12 · Collections (64–70s)
Save search as a collection. Click the collection → reruns instantly.

## 13 · Graph (70–76s)
Graph tab → new graph → add nodes from sessions → connect them.

## 14 · Graph search (76–82s)
Graph search bar. Non-matching nodes dim out.

## 15 · Graph multi-select (82–88s)
Ctrl+click multiple nodes. Their paragraphs open together in the editor.

## 16 · Compile (88–94s)
: → compile → exports a .docx file.

## 17 · New project (94–100s)
: → create project novel → new project created.

## 18 · Import file (100–106s)
Import a .md / .docx / .txt → new project from the file.

## 19 · Sessions (106–112s)
+ button → new session → rename inline.

## 20 · Duplicate & clusters (112–118s)
Right-click session → Duplicate.
Right-click → Move to cluster → session grouped.

## 21 · Reveal session (118–124s)
Right-click paragraph → Reveal session.
Flash in Explorer on the active session.

## 22 · Hashtags (124–130s)
Right-click paragraph → hashtags panel → add #tag.

## 23 · Quick notes (130–136s)
Ctrl+P → note modal → write → close.

## 24 · Paragraph versions (136–142s)
Explorer → clock icon on paragraph row → history → restore.

## 25 · Slash menu (142–148s)
/ → slash menu with all options.

## 26 · Help (148–152s)
F1 → full help in alphabetical order.

## 27 · Close (152–158s)
Fade to white. "Gombro" typewriter. Button: Download — 60-day free trial.
